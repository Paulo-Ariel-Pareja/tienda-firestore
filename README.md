TIENDA REALIZADA CON FIREBASE (FIRESTORE) Y ANGULAR by Paulo Ariel Pareja

El frontend es basico, ya que el objetivo de este ejercicio es la manipulacion de informacion con base de datos en tiempo real Firestore.

Para que el proyecto realice su tarea hay que configurar las credenciales de Firebase en ENVIRONMENT.TS

Los usuarios generados son con rol 'customer' por defecto. modificando el atributo de ROLE a 'editor' se agregan caracteristicas para la 
administracion de los articulos, siendo 'admin' el rol con todos los permisos.

tanto el rol 'editor' como 'admin' no tienen acceso a generar carrito.
Se adjunta archivo de configuracion de reglas para la seguridad de Firebase.
